# Assets Folder

Place your logo image file here.

**Recommended:**
- File name: `logo.png` or `logo.svg`
- Format: PNG (with transparency) or SVG (preferred for scalability)
- Size: At least 200x200px for best quality

The logo should match the BrandShift brand:
- Orange circle with white document and cloud icons
- Or your custom logo design

After adding your logo file, make sure the filename in `index.html` matches your file name.
